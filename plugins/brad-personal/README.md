# brad-personal

My personal Claude Code workflow helpers.

## Commands

- `/brad-personal:hello` - Test command to verify plugin installation

## Structure

```
brad-personal/
├─ .claude-plugin/
│  └─ plugin.json    # Plugin manifest
├─ commands/         # Slash commands
├─ agents/           # Agent configurations
├─ skills/           # Reusable skills
├─ hooks/            # Event hooks
└─ README.md
```

## Usage

After installing from the marketplace:

```
/brad-personal:hello
```
